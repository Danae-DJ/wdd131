Site Plan: Cycling Guide

Site Name

Cycling Choices was chosen as the site name because it reflects the core purpose of the website: to help users make informed decisions when choosing a bicycle for their needs.

Optional domain: cyclingchoices.com

Site Purpose

The site aims to educate visitors about bicycles, including their types, features, and best uses. It will also provide purchase links and pricing information to guide users in buying a suitable bicycle.

Scenarios

  • What is the best type of bicycle for urban commuting?
  • Which bicycles are better for mountain trails?
  • Where can I buy affordable bicycles?

Color Schema

The following colors will be used for the site:

  • #2C3E50 (Dark Blue): Used for headings and navigation background.
  • #E74C3C (Red): Used for call-to-action buttons and links.
  • #ECF0F1 (Light Gray): Used for the page background.

Typography

The following fonts will be used for the site:

  • Montserrat: Used for headings.
  • Roboto: Used for body text.
